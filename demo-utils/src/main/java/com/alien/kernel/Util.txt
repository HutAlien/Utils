GIT常用命令：
git pull --rebase origin master
回退commit：git reset --mixed HEAD~1
在本地添加tag标注：git tag  -a [tag名称]  -m"tag简单描述"
本地单个tag推送：git push origin [tag名称]
强推：git push origin --force
